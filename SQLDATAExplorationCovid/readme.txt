SQL Data Exploration on COVID-19 Dataset

Analyzed global COVID-19 data using advanced SQL queries, focusing on cases, deaths, and vaccination trends. Key insights include mortality rates, infection percentages, and vaccination impact by country and continent.

Description:

This project showcases SQL proficiency in data exploration and analysis. It includes:

1.Calculating death rates and infection percentages
2.Analyzing vaccination rollout using CTEs, JOINS, and temporary tables
3.dentifying global and regional COVID-19 trends
4.Creating views for streamlined insights

Technologies: SQL Server, Data Analysis, JOINs, CTEs, Temporary Tables, Data Visualization
